Algorithm Visualiser

This is a tool to let users simulate popular alogrithms on a graph. The website automatically generates a random connected graph once a user picks the number of nodes using the slider. User can select one of the algortihms from the list to see a step by step simualtion. A user can also pick a start node (where applicable) by clicking on one of the graph nodes.
